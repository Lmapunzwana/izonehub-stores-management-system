# Stores Management System

## Features Included
- Dashboard
- Items management
- Expected receipts
- Goods Received Note workflow
- Material requests
- Add item to request workflow
- Projects and project details
- Employees management
- Users administration (multi-role support)
- Returns workflow
- Discrepancies management
- Reports page
- Raw CSS styling (no Tailwind or component library)

## Run
npm install
npm run dev
